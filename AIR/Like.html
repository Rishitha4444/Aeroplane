LIKE OPERATOR

  • The LIKE operator is used in the WHERE condition to filter data based on some specific pattern.It can be used with numbers, string, or date values.

  • However, it is recommended to use the string values.

  • LIKE query is used to search for a particular pattern from the table rows and return the columns, which matches the pattern.

  • When you only know a fragment of a text value and need to get the details from the table. Then you can get the values that match the pattern mentioned by using the “LIKE” query in SQL.
