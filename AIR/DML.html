DML Commands

  • It consists of three different commands.

  • Data Manipulation Language (DML) commands in SQL deals with manipulation of data records stored within the database tables.

  • We can easily access, store, modify, update and delete the existing records from the database using DML commands.

  • There are three types in DML Commands : insert,update,delete.

INSERT – Used to insert new data records or rows in the database table.

UPDATE -Used to set the value of a field or column for a particular record to a new value.

DELETE – Used to remove one or more rows from the database table.
