LOGICAL OPERATORS

  • Logical operators are used to specify conditions in the structured query language (SQL) statement.

  • They are also used to serve as conjunctions for multiple conditions in a statement.

  • The following are the logical operators:

AND OPERATOR – This operator allows you to construct multiple conditions in the WHERE clause of an SQL statement. AND operator returns true if both expressions evaluate to true.

OR OPERATOR - This operator combines multiple conditions in an SQL statement’s WHERE clause. OR operator returns true if a least one expression evaluates to true.

NOT OPERATOR – NOT operator takes a single boolean as an argument and changes its value from false to true or from true to false.

DISTINCT - The SQL DISTINCT keyword is used in conjunction with the SELECT statement to eliminate all the duplicate records and fetching only unique records.

LIKE OPERATOR - LIKE operator compares a value to similar values using a wildcard operator.
